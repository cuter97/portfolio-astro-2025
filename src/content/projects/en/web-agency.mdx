# CIPRES AGENCY – Modern Web Agency Template

**CIPRES Agency** is a cutting-edge, professional website template designed for digital agencies, creative studios, and software houses. It represents a significant technical leap, utilizing the latest web technologies to provide a seamless, high-performance user experience.

The project focuses on **ultra-fast performance** and **sophisticated motion design**, combining Astro 5’s speed with React 19’s interactivity. Every interaction is enhanced by GSAP-powered animations and Lenis smooth scrolling, creating a premium "boutique" feel.

![Screenshot of CIPRES Agency website](/img/img-5.webp)

## Technical Excellence

Unlike traditional templates, CIPRES Agency is a **full-stack solution**:
- **Astro 5 + React 19**: Utilizing the newest "Islands Architecture" for selective hydration.
- **Tailwind CSS v4**: Leveraging the latest CSS-first engine for styling.
- **Server Actions**: Secure contact form handling with server-side validation via Zod.
- **Email Integration**: Built-in Resend API support for reliable communication.
- **Type-Safe Content**: Centralized management through `labels.ts` for easy internationalization and maintenance.

## Key Features

- **Performance First**: Optimized loading speeds and SEO out-of-the-box.
- **Advanced Motion**: Scroll-triggered timelines and professional transitions.
- **Ready-to-use Backend**: Functional contact system with rate limiting (Upstash Redis).
- **Adaptive Design**: Precision-crafted responsiveness for all devices.
- **Inclusive**: WCAG compliant components and accessible navigation.

The project was built with clean code and scalability in mind:
