# CIPRES AGENCY – Plantilla Moderna para Agencias Web

**CIPRES Agency** es una plantilla de vanguardia diseñada para agencias digitales, estudios creativos y software houses. Este proyecto representa un salto técnico significativo, utilizando las últimas tecnologías web para ofrecer una experiencia de usuario fluida y de alto rendimiento.

El enfoque principal del proyecto es el **rendimiento ultra rápido** y un **diseño de movimiento sofisticado**, combinando la velocidad de Astro 5 con la interactividad de React 19. Cada interacción está potenciada por animaciones GSAP y scroll suave (Lenis), logrando una estética "premium" y moderna.

![Captura de pantalla de CIPRES Agency](/img/img-5.webp)

## Excelencia Técnica

A diferencia de las plantillas tradicionales, CIPRES Agency es una **solución full-stack**:
- **Astro 5 + React 19**: Uso de la arquitectura de islas para una hidratación selectiva y eficiente.
- **Tailwind CSS v4**: Implementación del motor más reciente basado en CSS-first para estilos.
- **Server Actions**: Manejo seguro del formulario de contacto con validación del lado del servidor mediante Zod.
- **Integración de Email**: Soporte nativo para la API de Resend para asegurar la entrega de correos.
- **Contenido Tipado**: Gestión centralizada de textos a través de `labels.ts`, facilitando el mantenimiento y futuras traducciones.

## Características Principales

- **Rendimiento Optimizad**: Tiempos de carga mínimos y SEO configurado de base.
- **Movimiento Avanzado**: Timelines activados por scroll y transiciones profesionales.
- **Backend Listo para Usar**: Sistema de contacto funcional con límite de peticiones (Upstash Redis).
- **Diseño Adaptativo**: Responsividad pulida al detalle para móviles, tablets y escritorio.
- **Accesible**: Componentes que cumplen con los estándares WCAG y navegación accesible.

## Experiencia de Desarrollo (DX)

El proyecto fue construido pensando en la escalabilidad y el código limpio:
- **Arquitectura Modular**: Separación clara entre componentes de UI y secciones de página.
- **Sistema de Temas Global**: Personalización sencilla de colores y tipografía mediante variables CSS.
- **Gestión de SEO**: Sistema centralizado de meta-tags para títulos, descripciones y redes sociales.
- **Listo para Producción**: Pre-configurado para despliegue en Vercel, Netlify o Cloudflare.

## Resumen

**CIPRES Agency** es más que una simple plantilla; es una base robusta para el desarrollo web moderno. Logra el equilibrio perfecto entre la velocidad de un sitio estático y la funcionalidad de una aplicación dinámica, siendo la opción ideal para agencias que buscan demostrar su capacidad técnica y sensibilidad de diseño.
