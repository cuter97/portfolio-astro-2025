# Artesano del sur

Este proyecto es un catálogo digital para **'Artesano del Sur'**, una marca de productos artesanales.  
El sitio web está construido con **Astro** para optimizar el rendimiento, **Tailwind CSS** para estilizar rápidamente los diseños, y **GSAP** para añadir animaciones suaves que complementan la estética artesanal.  
El sitio es bilingüe y permite a los visitantes explorar piezas únicas hechas a mano.

![Captura de pantalla del sitio web Artesano del sur](/img/img-2.webp)

## Desafíos y Soluciones

- Implementé una estructura multilingüe accesible utilizando rutas dinámicas.
- Logré animaciones fluidas con GSAP sin comprometer el rendimiento.
- Aproveché Astro Islands para renderizar solo lo necesario y mejorar los tiempos de carga.

## Características

- Diseño responsivo con enfoque mobile-first
- Animaciones GSAP para transiciones suaves y atractivas
- Navegación bilingüe
- Secciones como historia, blog y catálogo de productos
