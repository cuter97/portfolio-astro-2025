# Olmo Studio

Olmo Studio es una plantilla profesional y de alto rendimiento para estudios digitales, construida para agencias, estudios creativos y profesionales que buscan liderar con impacto, rendimiento y un estilo visual cinematográfico. Diseñada con Astro 5 y React 19, esta plantilla combina diseños audaces y modernos con animaciones GSAP ultra suaves y una estructura de contenido robusta para guiar a los visitantes a través de tus servicios y proyectos sin esfuerzo.

![Captura de pantalla del sitio web de Olmo Studio](/img/img-7.webp)

## Desafíos y Soluciones

- **Rendimiento vs. Movimiento**: Logré velocidades de carga ultra rápidas mientras mantenía transiciones GSAP de nivel profesional y animaciones activadas por scroll mediante la estrategia de hidratación parcial de Astro.
- **Arquitectura Escalable**: Implementé un sistema de contenido centralizado usando un archivo `labels.ts` y configuraciones impulsadas por TypeScript, permitiendo actualizaciones fáciles en todas las páginas sin tocar el código principal.
- **Contacto Listo para Producción**: Desarrollé un formulario de contacto seguro con validación del lado del servidor usando Zod e integración con la API de Resend, incluyendo protección anti-spam opcional con Upstash Redis.
- **Experiencia de Portafolio Única**: Creé un efecto de apilamiento adhesivo de tarjetas para la vitrina de proyectos en escritorio, donde las tarjetas se apilan con animaciones de escala para una experiencia de navegación memorable.

## Características

- Rendimiento Ultra Rápido: Construido con Astro 5 para velocidades de carga óptimas y SEO de primer nivel.
- Diseño de Movimiento Suave: Transiciones profesionales con GSAP, animaciones activadas por scroll y desplazamiento suave con Lenis.
- Conjunto Completo de Páginas: Incluye Inicio, Trabajos (Portafolio), Estudio (Acerca de), Contacto, Servicios y páginas Legales, con páginas de detalle de proyectos individuales generadas desde Markdown.
- Configuración Tipada: Todos los datos del sitio (equipo, servicios, valores, testimonios) gestionados mediante TypeScript para personalización sin errores.
- Barra de Navegación Inteligente: Encabezado que se reduce y obtiene bordes redondeados al hacer scroll mediante animaciones GSAP.
- Diseño Responsivo: Meticulosamente optimizado para experiencias de visualización en móvil, tablet y escritorio.

## Mejoras Futuras

- Soporte Multi-idioma: Aprovechar la estructura existente de `labels.ts` para implementar internacionalización (i18n) completa.
- Integración con CMS Headless: Conectar las colecciones de contenido con plataformas como Sanity o Contentful para gestión de contenido no técnica.
- Analíticas Avanzadas: Integrar seguimiento de visualizaciones de proyectos e insights del comportamiento de visitantes.
