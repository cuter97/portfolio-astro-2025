# Larch Agency — Plantilla Moderna para Agencias de Marketing

Larch Agency es una plantilla web de alta gama para agencias de marketing y creativos, diseñada para estudios, agencias digitales y profesionales que desean mostrar su trabajo con impacto, claridad y rendimiento.

Diseñada con un lenguaje visual cinematográfico y un fuerte enfoque en movimiento y storytelling, Larch Agency combina layouts audaces, animaciones suaves con GSAP y una estructura de contenido limpia para guiar a los visitantes a través de servicios, proyectos y puntos de conversión sin esfuerzo.

![Captura de pantalla del sitio web de Larch](/img/img-6.webp)

## Descripción del Proyecto

Esta plantilla ofrece una presencia de agencia premium que se siente moderna, segura y lista para producción desde el primer día. Cada sección está cuidadosamente diseñada para comunicar valor, mostrar experiencia y convertir visitantes en clientes potenciales.

## Stack Tecnológico

- **Astro 5** — Framework principal para máximo rendimiento y arquitectura estático-primero
- **React Islands** — Interactividad selectiva para componentes dinámicos manteniendo el JavaScript del lado del cliente al mínimo
- **Tailwind CSS v4** — Estilos utility-first para tokens de diseño consistentes y layouts responsivos
- **TypeScript** — Configuración type-safe y comportamiento predecible de componentes
- **GSAP** — Animaciones profesionales e interacciones activadas por scroll
- **Lenis** — Experiencia de scroll suave
- **Resend** — Formulario de contacto seguro con entrega de email
- **Upstash** — Limitación de tasa del lado del servidor para protección anti-spam

## Aspectos Destacados de la Arquitectura

- **Enfoque performance-first:** Generación estática con interactividad delimitada
- **Movimiento como herramienta funcional:** Las animaciones GSAP mejoran el storytelling sin sacrificar rendimiento
- **Estructura escalable:** Componentes modulares diseñados para crecer con las necesidades del cliente
- **Listo para producción:** Incluye flujo de contacto, páginas legales y optimización SEO desde el inicio
- **Type safety:** TypeScript asegura comportamiento predecible y mantenimiento más fácil

## Filosofía de Desarrollo

Este proyecto refleja mi enfoque hacia el desarrollo frontend moderno:

- **Rendimiento sobre decoración** — Cada animación e interacción tiene un propósito
- **Composición limpia de componentes** — Reutilizable, mantenible y fácil de extender
- **Arquitectura estático-primero** — Tiempos de carga rápidos y SEO óptimo sin comprometer la interactividad
- **Calidad de grado comercial** — Construido para ser desplegado inmediatamente en proyectos reales de clientes

## Contexto

Larch Agency fue diseñada y desarrollada como una plantilla de grado comercial para agencias de marketing y creativas. Demuestra un equilibrio entre diseño cinematográfico, movimiento suave y código mantenible—probando que las experiencias visuales de alta gama no requieren sacrificar rendimiento o experiencia del desarrollador.

La plantilla es adecuada para proyectos comerciales reales, trabajo con clientes y agencias que construyen múltiples sitios web de presentación con estándares consistentes de calidad y rendimiento.
