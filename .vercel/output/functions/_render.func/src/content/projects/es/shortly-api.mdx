# Shortly API

Este proyecto fue un desafío de Frontend Mentor. El desafío implica integrar la API rel.ink en una interfaz de usuario web para crear URLs acortadas, que deben mostrarse de acuerdo al diseño propuesto. Tomé esta oportunidad para adquirir más experiencia con JavaScript asíncrono. Es uno de mis primeros proyectos y se completó en 2022.

![Screenshot de Shortly API website](/src/assets/img-3.webp)

## Desafíos y Soluciones

- Aprendí a trabajar con APIs y manejar operaciones asíncronas de manera efectiva.
- Diseñé la interfaz de usuario para cumplir con las especificaciones del desafío.

## Características

- Integración con la API rel.ink para acortar URLs
- Interfaz de usuario amigable para crear y gestionar URLs acortadas.

## Mejoras Futuras

- Mejorar la experiencia del usuario con características adicionales como análisis de enlaces y URLs acortadas personalizadas.
