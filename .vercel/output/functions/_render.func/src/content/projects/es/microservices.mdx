# Microservices
Este proyecto es parte de la cursada de microservicios en DevTalles Academy. Utiliza Nest.js para crear servicios escalables, Docker para contenerización, y NATS para comunicación asíncrona entre microservicios. Los servicios incluyen gestión de productos, pedidos, pagos, y autenticación.

## Características

- Microservicios comunicados via NATS
- Contenerización con Docker
- Gestión de productos, pedidos y pagos.

## Mejoras Futuras

- Crear un perfil administrador para agregar información dinámicamente.
- Mejorar la gestión de eventos y comunicación entre microservicios utilizando patrones de diseño de mensajería avanzados.
